Acha-Rapido
Agilidade Visual

Jogo de agilidade visual inspirado no clássico Lince, feito em HTML5 Canvas para rodar em qualquer celular ou computador, sem instalar nada.

## 🎯 Como jogar
1. Clique em Jogar
2. No topo aparece "ENCONTRE ESTE!" com um emoji
3. Você tem 30 segundos para achar e tocar no mesmo emoji no tabuleiro
4. Acerto = +1 ponto e novo tabuleiro
5. Erro = tela pisca vermelho e perde 1 segundo

## ♿ Acessibilidade - Modo Idoso e Criança
Pensado para idosos e crianças:
- Clique no botão Aa
- Arraste o slider:
    - Muito Pequeno: ~240 figuras
    - Pequeno: ~180 figuras
    - Normal: ~112 figuras
    - Criança: ~60 figuras grandes
    - Idoso: ~35 figuras GIGANTES

- Detecção de coordenadas
